#Aegis Network
## 
이 모듈은 네트워크 프로그램(서버, 클라이언트)에서 상호간 통신을 가능하게 지원하는 네트워크 기능을 제공합니다.

## Features
+ AegisNetwork
> Acceptor, Session, Packet

+ Aegis.Client
> Connector, Packet

##Supported Systems
+ Server side application
> Windows(32 bit, 64 bit) for .Net 4.5 or higher
    
+ Client side application
> Windows(32 bit, 64 bit) for .Net 3.5 or higher
> Mobile platform to support the Unity3D
